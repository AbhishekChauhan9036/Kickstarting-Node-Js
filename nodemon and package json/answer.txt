When would you use --save-dev and --save?
Ans=> --save is used when you want to add a package as a dependency in your project, 
meaning the package is required for the project to run properly. --save-dev is used 
when you want to add a package as a development dependency, meaning the package is 
only needed for development and testing, but not for the actual running of the project.
